# Sai Sudharsan IPL Analytics Dashboard

## Overview

This is an end-to-end Data Analytics project built using MySQL, Microsoft Excel, and Power BI to analyze the IPL performance of Sai Sudharsan.

The project demonstrates the complete analytics workflow from data collection and cleaning to SQL-based analysis and dashboard visualization. Data was collected from Cricbuzz and IPLT20, organized in Excel, converted into CSV format, analyzed using MySQL, and visualized through an interactive Power BI dashboard.

## Data Sources

- Cricbuzz
- IPLT20 (Official IPL Website)

## Tools Used

- MySQL
- Microsoft Excel
- Power BI

## Project Workflow

- Collected IPL batting statistics from Cricbuzz and IPLT20.
- Organized and cleaned the data in Excel.
- Converted the dataset into CSV format.
- Imported data into a MySQL database.
- Performed 40+ SQL queries covering season-wise, opponent-wise, venue-wise, and batting performance analysis.
- Created SQL Views for reporting and analysis.
- Built an interactive Power BI dashboard for visualization and insight generation.

## Analysis Performed

- Data Validation & Quality Checks
- Career Statistics Analysis
- Season-wise Performance Analysis
- Opponent-wise Analysis
- Venue-wise Analysis
- Home vs Away Comparison
- Match Result Analysis
- Batting Position Analysis
- Dismissal Analysis
- Boundary Analysis

## Dashboard Features

- KPI Cards (Runs, Average, Strike Rate, Highest Score)
- Runs by Season
- Runs by Opponent
- Runs by Venue
- Runs by Batting Position
- Boundary Analysis
- Dismissal Analysis
- Home vs Away Comparison
- Interactive Season Filter
